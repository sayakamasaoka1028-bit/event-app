# 📅 行事管理アプリ（LINE通知付き）

## 概要
家族で行事を共有し、LINEで通知・確認ができるWebアプリです。

## 主な機能
- 行事の登録・編集・削除
- 行事一覧表示
- LINE通知（自分・家族）
- 「確認しました」ボタン
- 誰が確認したか表示

## 使用技術
- Laravel
- PHP
- SQLite
- LINE Messaging API

## 工夫した点
- LINE APIを利用し、実生活で使えるアプリを作成
- Webhookによるリアルタイム処理
- 家族間での共有を意識した設計
